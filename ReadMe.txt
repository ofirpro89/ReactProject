פרויקט React – ניהול כרטיסים
תיאור הפרויקט
זהו פרויקט ריאקט הכולל מערכת לניהול כרטיסים לפי הרשאות משתמשים. הפרויקט נבנה בשלבים, עם תכנון מראש של מבנה העמודים והפונקציונליות הנדרשת לכל סוג משתמש – רגיל, עסקי ואדמין.

תהליך העבודה
1. תכנון מבנה העמודים
השלב הראשון היה להגדיר את מבנה האפליקציה – אילו עמודים יהיו, ומה תפקידו של כל עמוד. התכנון כלל שלושה מסכים עיקריים:

עמוד הרשמה/התחברות

מסך הבית

אזור אישי

2. עמוד הרשמה / התחברות
זהו העמוד הראשון שנבנה בפרויקט. הוא מאפשר הרשמה והתחברות על פי סוג המשתמש:

יוזר רגיל – גישה לצפייה בכרטיסים בלבד

יוזר עסקי (Business) – גישה לצפייה ולניהול כרטיסים

אדמין (Admin) – גישה מלאה לכלל הנתונים והיכולות במערכת

המטרה הייתה לבנות לוגיקת הרשאות בסיסית שתשלוט על מה שכל משתמש יכול לראות ולעשות.

3. מסך הבית
במסך זה מוצגים כלל הכרטיסים הקיימים במערכת. זהו המסך הראשי שזמין לכל המשתמשים, והוא מאפשר לעיין בכרטיסים ללא צורך בהרשאות מיוחדות.

בעתיד ניתן להרחיב את המסך הזה לפילטור, חיפוש, מיון ועוד.

4. אזור אישי (Dashboard)
עמוד זה מיועד בעיקר למשתמשים עסקיים. כאן הם יכולים:

לבנות כרטיסים חדשים

לנהל כרטיסים קיימים

לערוך או למחוק כרטיסים שיצרו

הגישה לעמוד זה מתאפשרת רק למשתמשים עסקיים דרך מערכת ניהול הרשאות פנימית.

טכנולוגיות עיקריות
React

React Router

Local Storage / Session Storage

CSS / Styled Components
(ניתן להוסיף ספריות נוספות אם השתמשת)

הערות נוספות
הפרויקט נבנה בשאיפה לקוד מודולרי ונקי

נעשה שימוש בנתיבי גישה מבוססי הרשאות

תכנון עתידי כולל הוספת API חיצוני, אימות בטוקנים (JWT), והרחבת אפשרויות הניהול
